# AUBus - Carpool Application
### Prerequisites
- Python 3.7 or higher
- Windows/Linux/Mac OS

### Installation Steps

1. *Install Required Dependencies*:
bash
pip install PyQt5 selenium pywhatkit requests webdriver-manager

### Running the Application

1. *Start the Server* (First Terminal):
bash
cd path/to/project/folder 
python server.py

2. *Start the Client* (Second Terminal - New Terminal Window):
bash
cd path/to/project/folder
python main.py


### Important Notes
- *Both terminals must be open simultaneously*
- *Keep server.py running in the background*
- *Make sure you're in the correct directory containing the project files*
- *Internet connection required for maps and weather features*

The application window will open automatically after running python main.py.
